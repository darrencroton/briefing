# Series Examples

This directory contains tracked example series configuration.

These files are examples only. `briefing` does not load them as live user configuration.

Create your real local series files under `user_config/series/`, either by:

- copying and adapting one of the examples here, or
- running `uv run briefing init-series`

If you are new to the app, read these first:

- [`../../../docs/setup-and-configuration-walkthrough.md`](../../../docs/setup-and-configuration-walkthrough.md)
- [`../../../docs/source-guides/README.md`](../../../docs/source-guides/README.md)
